## Swift로 작성한 첫번째 프로그램

- Swift는 코드 한 줄로 `Hello, world!`를 출력할 수 있다.
- 예시의 한 줄의 코드가 완전한 프로그램이다. 
- 전역으로 작성된 코드는 프로그램의 시작점이므로 main() 함수가 필요 없다. 
- 코드의 끝마다 세미콜론(`;`)을 붙일 필요가 없다.

~~~swift
print("Hello, world!")
// Hello, world!
~~~
